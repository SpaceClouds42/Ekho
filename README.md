# Ekho

## Usage

### 1. Add to your project
```groovy
// build.gradle
repositories {
    maven { url = "https://jitpack.io" }
}

dependencies {
    // Ekho
    modImplementation "com.github.SpaceServe:Ekho:0.3.0"
    include "com.github.SpaceServe:Ekho:0.3.0"
}
```
